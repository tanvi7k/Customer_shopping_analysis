**Project 1: Sales & Customer Behavior Analysis Dashboard**
**Project Objective**

The objective of this project is to analyze sales and customer behavior data to identify trends, patterns, and key performance indicators (KPIs) that help improve business decision-making and drive sales growth.

**Tools & Technologies Used**

Microsoft Excel

Power BI

Data Cleaning & Transformation

Data Visualization

Dashboard Design

Exploratory Data Analysis (EDA)

**Dataset Description**

The dataset contains historical sales and customer information, including:
Order details
Customer demographics (age, gender, location)
Sales valus
Product categories
Sales channels
Order status

**Data Cleaning & Preparation**

Removed missing and duplicate values

Verified data consistency and accuracy

**Data Analysis Process**

Defined business objectives and KPIs
Cleaned and transformed raw dat
Created pivot tables and measures
Built interactive visualizations in Power BI
Designed a dynamic dashboard using slicers and filters
Analyzed trends and patterns
Derived actionable insights

**Dashboard Features**

Interactive Power BI dashboard
Clear visual representation of trends and insights

**Business Impact**

Helps stakeholders understand customer behavior
Identifies high-value customer segments
Supports data-driven marketing strategie
Improves sales planning and inventory decisions

**Conclusion**

This project demonstrates strong skills in data analysis, visualization, and business insight generation. The interactive dashboard enables stakeholders to monitor performance, identify growth opportunities, and make informed decisions to improve overall business outcomes.
